# Premium Loading Spinner Using HTML & CSS

A modern and lightweight loading spinner built with pure HTML & CSS.  
Designed with smooth animations, scalable SVG structure, and customizable styling for seamless integration into modern web projects.

---

## ✨ Overview

This project showcases a premium animated loading spinner component built without JavaScript.  
It focuses on performance efficiency, clean design, and scalable structure suitable for professional UI implementation.

---

## 🚀 Features

- Smooth CSS animations
- SVG-based scalable structure
- Lightweight and fast rendering
- Fully responsive
- Easy customization using CSS variables
- Modern minimal design

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Animations, Keyframes, Variables)
- SVG

---

## 📂 Use Case

Perfect for:
- Landing pages
- Web applications
- Dashboard loading states
- Portfolio projects
- UI component libraries

---

## 🎯 Project Goal

To demonstrate advanced CSS animation techniques and the ability to create high-quality UI components without external libraries.
